## Setup

1. Make sure you are using Python 3 and have the following installed:
  * numpy
  * nltk
  * scipy
  * sklearn


2. Run `naive_bayes/NaiveBayes.py` to run the Naive Bayes classifier

3. Run `svm/utils.py` to run the SVM and the MaxEnt classifiers
